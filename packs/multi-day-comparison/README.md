# Multi-Day Comparison Pack

**Audience:** SA running a full customer POC engagement  
**Time budget:** 2–3 days  
**Goal:** Execute every topology and failure scenario, collect comprehensive evidence, produce a completed scorecard and executive readout.

---

## What You'll Do

- **Day 1:** Stand up all topologies, run baselines on each, verify observability
- **Day 2:** Execute all failure and operations scenarios, collect evidence packs
- **Day 3:** Cross-run comparison, fill scorecard, draft executive readout

## What You'll Produce

- Baseline + failure evidence for every scenario on every platform
- A completed POC scorecard with real data
- A draft executive readout ready for customer presentation
