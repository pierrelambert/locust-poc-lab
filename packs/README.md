# Motion Packs

Pre-built playbooks that bundle the right guides, runbooks, templates, and make targets for a specific SA motion. Pick the pack that matches your situation.

---

| Pack | Audience | Time | Description |
|------|----------|------|-------------|
| [SA Enablement](sa-enablement/) | New SA ramping up | ~4 hours | Self-study path through every lab guide and scenario runbook |
| [Customer Demo](customer-demo/) | SA doing a live demo | 60 minutes | Scripted live demo: baseline → primary kill → show evidence |
| [Multi-Day Comparison](multi-day-comparison/) | SA running a full POC | 2–3 days | All topologies, all failure scenarios, full evidence collection |
| [Executive Readout](executive-readout/) | SA preparing deliverables | 2–4 hours | Gather evidence → scorecard → executive readout → recommendations |

## How to Use

1. **Choose a pack** based on your motion.
2. **Open the pack's README** for context (who it's for, what you'll produce, time budget).
3. **Follow the CHECKLIST** step by step — every item links to the exact file or `make` target you need.

## Prerequisites

All packs assume you have completed the initial setup described in [`FIRST_30_MINUTES.md`](../FIRST_30_MINUTES.md) or at minimum run `make setup`.
