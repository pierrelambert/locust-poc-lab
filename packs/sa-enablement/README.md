# SA Enablement Pack

**Audience:** New Solutions Architect ramping up on the Locust POC Lab  
**Time budget:** ~4 hours (self-paced)  
**Goal:** Build hands-on fluency with every topology, scenario, and deliverable in the lab so you can confidently run customer-facing POCs.

---

## What You'll Learn

1. How to stand up Redis Enterprise and OSS Redis topologies (Docker, k8s, VM paths)
2. How to run each failure and operations scenario using the runbooks
3. How the observability stack captures evidence (Prometheus, Grafana, canary writer)
4. How to interpret results and fill out a POC scorecard
5. How to compare runs across platforms using the cross-run comparison tooling

## What You'll Produce

- A completed baseline + primary-kill run on at least one topology
- A filled-in sample scorecard
- Familiarity with every scenario runbook
